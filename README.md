# SwiftUi-Learning

SwiftUi-Learning is a small SwiftUI practice project based on a Landmarks-style app. It recreates a simple landmark detail screen with a map, circular image, title, location metadata, and description layout.

## Features

- SwiftUI app entry point with `@main`.
- Landmark detail UI built with `VStack`, `HStack`, `Spacer`, `Divider`, and modifiers.
- MapKit integration through SwiftUI `Map`.
- Turtle Rock image clipped into a circle with border and shadow.
- Xcode previews for the main view, map view, and circular image view.
- Default unit test and UI test targets generated by Xcode.

## Tech Stack

- Swift
- SwiftUI
- MapKit
- XCTest
- Xcode project structure

## Project Structure

```text
Landmarks/
├── Landmarks
│   ├── LandmarksApp.swift   # SwiftUI app entry point
│   ├── ContentView.swift    # Main landmark detail screen
│   ├── MapView.swift        # MapKit/SwiftUI map component
│   ├── CircleImage.swift    # Circular landmark image component
│   └── Assets.xcassets      # App icons, colors, and Turtle Rock image
├── LandmarksTests           # Unit test target
├── LandmarksUITests         # UI test target
└── Landmarks.xcodeproj
```

## Main Screen

The current screen shows:

- A map centered on Turtle Rock coordinates.
- A circular Turtle Rock image overlaid below the map.
- The title `Turtle Rock`.
- Location details for Joshua Tree National Park, California.
- A placeholder description section.

## Getting Started

1. Clone the repository.
2. Open `Landmarks/Landmarks.xcodeproj` in Xcode.
3. Select an iOS simulator or device.
4. Build and run the `Landmarks` target.

## Notes

This repository is a SwiftUI learning workspace. It is intentionally small and focused on core SwiftUI layout, previews, image styling, and MapKit basics.
